I'm a Senior QA Automation Engineer (SDET) with over 6 years of hands-on experience designing and scaling test automation systems that empower engineering teams to ship faster and with confidence.

My expertise spans the full test spectrum — from UI automation with Playwright, Cypress & Selenium, to API testing, mobile automation with Appium, and deep CI/CD pipeline integration with GitLab CI and GitHub Actions.

I have a track record of migrating legacy Selenium suites to Playwright, building component-based Page Object Model frameworks from scratch, and dramatically reducing flakiness and execution time in large enterprise test suites.

My work spans high-impact clients including The Walt Disney Company, where I delivered robust automated testing infrastructure for critical web applications.
